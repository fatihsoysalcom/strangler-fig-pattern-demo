# Strangler Fig Pattern Demo

This Python script demonstrates the Strangler Fig pattern. A facade intercepts requests, routing some to a new component while others still go to the legacy system, simulating a gradual migration.

## Language

`python`

## How to Run

Run this script using Python: `python strangle_fig_demo.py`
